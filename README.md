# 坦克大战

一个使用 HTML5 Canvas 和原生 JavaScript 开发的经典坦克大战游戏。

## 在线试玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格键 | 射击 |
| P | 暂停/继续 |

## 游戏玩法

- 控制绿色坦克消灭红色敌方坦克
- 击毁坦克得 100 分
- 砖墙可以被子弹击毁，钢墙无法破坏
- 每关需要消灭指定数量的敌人才能进入下一关
- 关卡越高，敌人速度越快、生成越快
